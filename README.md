# Features

* Calculate Birth Day From NIC

* Calculate Age (______years _______months ________days)

* Calculate Lived Days (_____days)

* Gender

* Displaye Birth Year type (Leap Year or NOT)

# Requirements

Python 3.9.7 Download
  * pyrogram
  * datetime
  * pytz

# Usage

```bash
python bot.py
```
